# AlexanderAEP

Componente que muestra una tarjeta con un avatar, un título y una descripción.

#### Ejemplo

<Canvas of={AlexanderAEP.Primary} />

## Uso

```tsx
import AlexanderAEP from 'milibreria';

function Example() {
  return (
    <AlexanderAEP
      title="Ale Allen"
      description="The Fastest Man Alive"
      avatar="https://example.com/Ale-avatar.jpg"
    />
  );
}

export default Example;
